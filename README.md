# XStreamify

Stream, Connect, Personalize: A Video Platform with Social Features and Smart Recommendations

</div>

XStreamify is a modern video streaming platform that blends seamless content delivery with social engagement. it enables users to manage playlists, interact through subscriptions, comments, and likes, and enjoy personalized recommendations shaped by user preferences.

## Key Features

- **Personalized Recommendations**: Feed control lets users shape their content discovery by adding interest tags and filtering unwanted topics.
- **Video Streaming & Playlists**: High-quality streaming with easy playlist creation and management.
- **Social Interactions**: Real-time engagement through subscriptions, comments, and likes to foster community.
- **Content Safety**: Parental Control system to restrict categories and monitor recommendations.
- **User-Friendly Design**: Clean interface with dark mode support for better accessibility.

### Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/saidinesh49/XStreamify.git
   cd XStreamify
   ```
2. Run the command
   ```bash
   npm run dev
   ```
